# Roadmap

This roadmap captures the intended sequence of product and workflow milestones.

## Milestones

### v0.1 - Project Setup

- Initialize Next.js project.
- Establish core app structure.
- Create first homepage modules.

### v0.2 - GitHub + Vercel

- Connect repository to GitHub.
- Connect GitHub to Vercel.
- Establish automatic deployment from `main`.

### v0.3 - AI Translate

- Add `/translate` route.
- Add `/api/translate` server route.
- Integrate OpenAI API.
- Support Chinese to Japanese and Japanese to Chinese text translation.

### v0.4 - Development Workflow

- Standardize development path as `~/Projects/chj-homepage`.
- Use GitHub as the single source of truth.
- Define iMac / MacBook synchronization workflow.
- Initialize documentation system.

### v0.5 - Translate Polish

- Improve Translate UI.
- Add kana support.
- Add explanations.
- Add copy-to-clipboard.
- Add read-aloud support.

### v0.6 - Translate History / Favorites

- Add translation history.
- Add favorite translations.
- Add persistence strategy.

### v0.7 - OCR

- Add image-based text extraction.
- Connect OCR output to Translate.

### v0.8 - Speech

- Add speech input.
- Add speech-to-text.
- Add read-aloud and listening workflows.

### v1.0 - AI Translate Complete

- Stable text, OCR, speech, history, favorites, copy, read-aloud, and explanation workflows.
- Production-ready Translate module.

## Future Modules

### Pony

Personal records, care logs, growth tracking, and related dashboards.

### Dashboard

Central overview of personal modules, schedules, and operational status.

### World Cup

Tournament tracking, schedules, rankings, and personal viewing tools.

### Finance

Personal finance dashboard, market watchlists, and reporting.

### Admin

Internal control center for project operations, diagnostics, and maintenance.
